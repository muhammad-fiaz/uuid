# UUID v6 (Reordered Time)

UUID v6 is the same as v1 but with time fields reordered for better sortability.

## Usage

```c
#include <uuid/uuid.h>

uuid_v1_state_t state;
uuid_t uuid;
char buf[UUID_STRING_LEN];

uuid_v6_init(&state);
uuid_v6(&state, &uuid);
uuid_format(&uuid, buf, sizeof(buf));
```

## Properties

- Version: 6
- Variant: RFC 4122 (10xx)
- Timestamp: Same as v1 (Gregorian epoch, 100-ns)
- Monotonic: Yes
- Sortable: Yes (time-ordered)

## v1 vs v6

| Property | v1 | v6 |
|----------|----|----|
| Time field order | time_low, time_mid, time_high | time_high, time_mid, time_low |
| Sortable | No | Yes |
| Compat with v1 | Yes | Yes |
| RFC 9562 | Legacy | Preferred |
