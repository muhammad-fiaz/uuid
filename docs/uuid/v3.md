# UUID v3 (MD5 Name-based)

UUID v3 generates a deterministic UUID from a namespace UUID and a name, using MD5.

## Usage

```c
#include <uuid/uuid.h>

uuid_t uuid;
char buf[UUID_STRING_LEN];

// Using well-known namespace
uuid_v3_ns(UUID_NS_DNS, "example.com", 11, &uuid);
uuid_format(&uuid, buf, sizeof(buf));
// e.g. "9073926b-929f-31c2-abc9-fad77ae3e8eb"

// Using custom namespace
uuid_t custom_ns;
uuid_v4(&custom_ns);
uuid_v3(&custom_ns, "my-name", 7, &uuid);
```

## Properties

- Version: 3
- Variant: RFC 4122 (10xx)
- Hash: MD5 (128 bits)
- Deterministic: Yes (same namespace + name = same UUID)
- Sortable: No

## Well-Known Namespaces

| Namespace | UUID |
|-----------|------|
| DNS | `6ba7b810-9dad-11d1-80b4-00c04fd430c8` |
| URL | `6ba7b811-9dad-11d1-80b4-00c04fd430c8` |
| OID | `6ba7b812-9dad-11d1-80b4-00c04fd430c8` |
| X500 | `6ba7b814-9dad-11d1-80b4-00c04fd430c8` |
