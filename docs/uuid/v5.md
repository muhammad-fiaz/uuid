# UUID v5 (SHA-1 Name-based)

UUID v5 generates a deterministic UUID from a namespace UUID and a name, using SHA-1.

## Usage

```c
#include <uuid/uuid.h>

uuid_t uuid;
char buf[UUID_STRING_LEN];

// Using well-known namespace
uuid_v5_ns(UUID_NS_DNS, "example.com", 11, &uuid);
uuid_format(&uuid, buf, sizeof(buf));
// e.g. "2ed6657d-b7e0-4b54-9bb7-2e8b9e2c3d1a"

// Using custom namespace
uuid_t custom_ns;
uuid_v4(&custom_ns);
uuid_v5(&custom_ns, "my-name", 7, &uuid);
```

## Properties

- Version: 5
- Variant: RFC 4122 (10xx)
- Hash: SHA-1 (160 bits, truncated to 128)
- Deterministic: Yes (same namespace + name = same UUID)
- Sortable: No

## v3 vs v5

| Property | v3 | v5 |
|----------|----|----|
| Hash | MD5 | SHA-1 |
| Security | Lower | Higher |
| Speed | Faster | Slightly slower |
| Recommendation | Legacy | Preferred |
